AquaSense AI

Django Machine Learning Water Quality Analysis IoT Integration Gradient Boosting Ensemble Models Environmental Tech API Integration

Project Overview

AquaSense AI is a sophisticated water quality analysis platform built with Django that employs ten different machine learning models, including Linear Regression, Polynomial Regression, SVR, Decision Trees, Random Forest, Gradient Boosting, and other ensemble models.

Features

  • Analysis of water parameters through manual entry of seven parameters (temperature, dissolved oxygen, pH, conductivity, BOD, nitrate, and total coliform)
  • IoT sensor data integration for four key parameters
  • Primary GBR model for predictions with ensemble assessment from all ten models
  • Comprehensive suggestion system offering remediation methods through:
    • Ayurvedic treatments
    • Scientific interventions
    • Cross-verification techniques
  • User-friendly interface with dedicated sections explaining challenges, vision, and mission
  • Firebase-connected contact system for user inquiries and feedback

Live Dashboard

Technical Implementation

The application uses Django for the backend, with a PostgreSQL database to store water quality data and prediction results. The machine learning models are implemented using scikit-learn and are integrated into the Django application using joblib for model serialization.
